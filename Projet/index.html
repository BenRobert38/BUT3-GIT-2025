FAQ

Comment utiliser Git ?

Git permet de suivre les modifications dans un projet collaboratif en permettant de gérer différentes versions d'un même fichier.

Pourquoi utiliser un gestionnaire de versions ?

Un gestionnaire de versions comme Git permet de travailler de manière efficace en équipe et de garder un historique de toutes les modifications apportées à un projet.

Comment créer un dépôt Git local ?

Pour créer un dépôt Git local, il suffit de naviguer dans un répertoire et de lancer la commande git init.
